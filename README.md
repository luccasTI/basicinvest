# basicinvest
Desafio proposto pelo curso Santander - Excel com Inteligência Artificial da DIO
O objetivo do desafio foi de criar uma **planilha básica para investimento, mais especificamente em FIIs**. Foram utilizadas algumas técnicas básicas de excel como, formatação condicional, validação de dados, função procv, colunas-chave.
A maior preocupação na criação da planilha foi a elaboração de um sistema que funcionasse de maneira automática, fácil de entender e que não precisasse que o usuário inserisse muitas informações.
